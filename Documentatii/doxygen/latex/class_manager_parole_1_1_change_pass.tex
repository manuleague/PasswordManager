\doxysubsection{Detailed Description}
Provides a user interface for setting or changing the master password. 

This form handles two scenarios\+:
\begin{DoxyEnumerate}
\item First-\/time setup\+: When no master password exists
\item Password change\+: When updating an existing master password
\end{DoxyEnumerate}

In password change scenarios, it also migrates encrypted data from the old password to the new password. The new password is stored in the Windows Registry along with a verification token.
