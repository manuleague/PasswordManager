\doxysubsection{Detailed Description}
Represents a single password entry in the password manager system. 

This class models all aspects of a stored credential including\+:
\begin{DoxyItemize}
\item Associated website/service (Url)
\item Username/\+Login ID (User\+Id)
\item Encrypted password (Encrypted\+Password)
\item Optional notes (Remarks)
\item Unique identifier (Id) for database operations 
\end{DoxyItemize}
